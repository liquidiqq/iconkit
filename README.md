## Installation

Install SvelteKit

```bash
npm create svelte@latest my-app
```

Install Tailwindcss (optional)

```bash
npx svelte-add@latest tailwindcss
```

Install Iconkit

```bash
npm install -D @liquidiqq/iconkit
```

## Usage

```html
<script>
	import { Icon } from '@liquidiqq/iconkit';
</script>

<Icon name="face-smile"></Icon>
```
